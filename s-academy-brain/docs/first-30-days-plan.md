# First 30 Days Plan

## Goal

Establish the foundational infrastructure for S.Academy's Knowledge Operating System.

## Success criteria

- Foundational Git repository exists.
- Knowledge Asset Repository Template exists.
- Introduction to Multifamily is migrated.
- Halo Overview is migrated.
- Repository management workflow is defined.
- Initial Client Intelligence structure is identified.
- Provider-neutral AI strategy is documented.

## Week 1: Repository Foundation

- Confirm repo structure.
- Add README, learn.md, and guidelines.
- Create courses, shared, templates, docs, and exports folders.
- Define metadata and naming standards.

## Week 2: Template System

- Build reusable course template.
- Build knowledge asset template.
- Build question bank template.
- Build prompt template.
- Build rubric and schema files.

## Week 3: First Asset Migration

- Migrate Introduction to Multifamily.
- Migrate Halo Overview.
- Validate course metadata.
- Add question banks.
- Add system and evaluation prompts.

## Week 4: Governance and Expansion

- Define update workflow.
- Define review process.
- Identify Client Intelligence use cases.
- Document provider-neutral AI considerations.
- Prepare v1 handoff.
