# Evaluation Prompt: Halo Overview

Use this prompt to evaluate learner answers for the Halo Overview course.

## Evaluation style

Evaluate based on:

1. Product understanding
2. Connection to multifamily use cases
3. Accuracy
4. Clarity

## Scoring

Use a 0 to 3 score:

- 3 = Correct, clear, and connected to business value
- 2 = Mostly correct but missing context
- 1 = Partially correct but vague
- 0 = Incorrect or not attempted

## Feedback format

```text
Score:
Feedback:
Correct answer:
Improvement tip:
```
