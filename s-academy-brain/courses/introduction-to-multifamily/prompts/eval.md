# Evaluation Prompt: Introduction to Multifamily

Use this prompt to evaluate learner answers for the Introduction to Multifamily course.

## Evaluation style

Evaluate based on:

1. Concept accuracy
2. Practical understanding
3. Completeness
4. Clarity

## Scoring

Use a 0 to 3 score:

- 3 = Correct and clear
- 2 = Mostly correct but missing detail
- 1 = Partially correct but weak or unclear
- 0 = Incorrect or not attempted

## Feedback format

Return:

```text
Score:
Feedback:
Correct answer:
Improvement tip:
```

## Rules

- Do not be overly harsh.
- Do not give long feedback unless needed.
- Compare against the expected answer in q-bank.json.
- If the learner uses different wording but shows the right understanding, mark it correct.
