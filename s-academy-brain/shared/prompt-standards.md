# Prompt Standards

## Purpose

Prompt files define how AI assistants should use course and knowledge content.

## Standards

- Keep prompts provider-neutral where possible.
- Separate learning assistance from evaluation.
- Do not include API keys or private configuration.
- Use clear role, behavior, scope, and output rules.

## Required prompt files per course

```text
prompts/system.md
prompts/eval.md
```

## Optional prompt files

```text
prompts/tutor.md
prompts/quiz-generator.md
prompts/summary-generator.md
```
